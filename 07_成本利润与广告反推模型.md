# 07_成本利润与广告反推模型

> 本阶段目标：判断产品在目标价格带下是否有利润空间，以及广告是否具备启动可能。  
> 红线：没有 BOM，不得输出完整利润结论；没有包装后尺寸重量，不得输出 FBA 后净利。

---

## 1. 成本利润模型定位

市场有机会，不代表产品能做。  
最终必须回答：

```text
哪个价格带能活？
哪个价格带只是冲量？
广告 ACOS 容忍上限是多少？
CPC 高于多少就不能打？
CVR 低于多少就必须停？
需要多少自然单占比才能盈利？
是否存在卖得越多亏得越多的风险？
```

---

## 2. 成本项拆分

| 成本项 | 说明 |
|---|---|
| BOM / 出厂价 | 工厂报价，币种统一 |
| 包材 | 彩盒、说明书、贴纸、保护袋 |
| JAN / 标签 | 单独计入 |
| 头程物流 | 按体积重或实际重量摊 |
| 关税/消费税 | 按站点口径 |
| FBA 配送费 | 按包装后尺寸重量 |
| 销售佣金 | 按类目费率 |
| Coupon 成本 | 折扣和相关成本 |
| 广告成本 | CPC × 点击数 / CVR |
| 退货损耗 | 按类目退货率 |
| 不良品损耗 | 按供应链不良率 |
| Vine / 测评摊销 | 新品启动期摊销 |
| 模具费 | 单独列，不得误入单件成本 |
| 汇率 | RMB/JPY 或 USD/JPY 固定口径 |

---

## 3. 成本口径红线

| 风险 | 处理 |
|---|---|
| BOM 不含包装 | 包装单独计入 |
| 包装、配件重复计价 | 拆分明细 |
| 模具费混入单件成本 | 单独列为一次性费用 |
| 币种混乱 | 统一换算 |
| 尺寸重量未确认 | 不输出 FBA 后净利 |
| Coupon 只算折扣不算实际到手价 | 按到手价建模 |
| 广告成本只按 ACOS 不按 CPA | 必须同时看 CPA、CPC、CVR |

---

## 4. 基础公式

### 单件净利

```text
单件净利 = 到手价
          - 销售佣金
          - FBA配送费
          - 头程物流
          - BOM
          - 包材
          - Coupon成本
          - 退货损耗
          - 广告成本
          - 其他成本
```

### 可承受 CPA

```text
可承受 CPA = 到手价
            - 销售佣金
            - FBA配送费
            - 头程物流
            - BOM
            - 包材
            - 退货损耗
            - 目标净利
```

### 可承受 CPC

```text
可承受 CPC = 可承受 CPA × CVR
```

### 盈亏平衡 ACOS

```text
盈亏平衡 ACOS = 可承受广告成本 / 到手价
```

## 5. 必做模型

| 模型 | 口径 |
|---|---|
| 保守模型 | 高 CPC、低 CVR、低售价、高成本 |
| 基准模型 | 市场中位口径 |
| 乐观模型 | 高 CVR、较好售价、成本可控 |

## 10. AI 可自行判断

| 判断项 | 说明 |
|---|---|
| 市场价格带 | 根据有效样本计算 |
| 竞品到手价分布 | 标价、Coupon、单件价 |
| 理论可承受成本区间 | 根据目标售价反推 |
| 广告 CPA/CPC 假设 | 根据 CVR 假设反推 |
| 利润敏感点 | 哪个成本项最影响利润 |
| FBA 尺寸风险 | 基于预估尺寸提示风险 |

---

## 11. 必须补数

| 数据 | 不补的影响 |
|---|---|
| BOM | 不得输出完整利润结论 |
| 包装后尺寸重量 | 不得输出 FBA 后净利 |
| 头程口径 | 到仓成本不可信 |
| 是否含包装 | 成本可能漏算 |
| 是否有模具费 | 回本测算不完整 |
| 目标售价 | 无法判断利润是否符合业务目标 |
| 广告预算 | 无法制定启动路径 |

## 13. 判断门

| 条件 | 处理 |
|---|---|
| 成本、FBA、售价完整 | 可输出财务结论 |
| 缺 BOM | 只输出市场侧价格反推 |
| 缺包装尺寸重量 | 不输出 FBA 后净利 |
| 广告模型显示可承受 CPC 远低于市场 CPC | 降级或终止 |
| 所有价格带均亏损 | 终止或回到供应链降本 |
